1. 
Перейти в папку в которую хотите скопировать 
скоприровать с GitHub 
командой git clone ссылка(скопированная)
готово!

2. 
git add название репозитория 
git commit -m "какие изменения" название репозитория(файла)
git push (добавили на GitHub)(отрпавит всё что было сделанно в commit)

3.
git pull (получаем изменения с GitHub)

4.
git status:
состояние рабочего каталога, можно проверить индексацию изменений и увидеть
файлы которых нет на GitHub

git log:
выводит список коммитов к данному репозиторию (кто и когда внёс изменения)

git remote -v:
показывает список удалённых подключений с URL адресом всех подключений
